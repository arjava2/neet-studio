# 🎯 NEET Question Studio

AI-powered NEET question generator built with Streamlit and Google Gemini.

## ✨ Features

- 📝 Generate NEET-level MCQs from NCERT PDFs
- 🎯 6 question types (MCQ, Assertion-Reason, Match the Following, etc.)
- 🌐 English + Hindi bilingual output
- ✅ Answer verification with AI
- ✍️ Humanization to bypass AI detection
- 🖼️ Extract diagrams directly from NCERT PDFs
- 📁 Easy PDF upload interface

## 🚀 Live Demo

Try it here: [neet-studio.streamlit.app](https://neet-studio.streamlit.app)

## 🔑 Getting Started

1. Get free Gemini API key from [Google AI Studio](https://aistudio.google.com/apikey)
2. Enter API key in sidebar
3. Upload your NCERT PDFs
4. Select chapter and generate questions!
